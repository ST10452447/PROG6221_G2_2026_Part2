# PROG6221_G2_2026_Part2
# Cybersecurity Chatbot - Part 2 (GUI)

## Features
- 🎤 Voice greeting on startup
- 🖼️ ASCII art logo in GUI
- 🔒 75+ cybersecurity tips across 5 topics
- 🎲 Random responses (15+ per topic)
- 🧠 Memory (remembers name + favorite topic)
- 💬 Follow-up handling ("tell me more")
- 😊 Sentiment detection (worried/curious/frustrated)
- 🏗️ Clean OOP code with delegate

## How to Run
1. Open solution in Visual Studio 2022
2. Build (Ctrl+Shift+B)
3. Run (F5)
